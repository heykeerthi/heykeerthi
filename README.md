## how i work

- **systems over snippets.** if i do a task twice, it becomes a file an AI can run.
- **gates, not vibes.** posts, features, and campaigns ship only if they pass a written gate. "not yet" is the default answer.
- **AI is an intern-to-colleague upgrade path.** the more judgment you write down, the better it pushes back.
- **know what can't be vibecoded.** brand, trust, taste, sales, and relationships stay human. code aggressively around them; never pretend code solves them.
